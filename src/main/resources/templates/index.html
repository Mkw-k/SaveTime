How It Works

1. 재생목록 업데이트

내 재생목록 업데이트 버튼을 눌러 재생목록을 업데이트 해주세요.

2. 재생목록 불러오기

기존에 검색하여 저장되어있는 재생목록은 따로 재생목록 업데이트를 진행하지 않아도 불러오기가 가능합니다. 하지만 비공개 재생목록의 경우에는 영상보기 페이지로 이동이 불가합니다. 비공개 재생목록의 경우 영상보기 페이지로 이동하시려면 내 재생목록 업데이트를 한번 진행 하신 후에 이동이 가능합니다.

3. 재생목록 검색

한번 업데이트 해놨던 재생목록들은 검색하기 기능을 통해 쉽게 검색이 가능합니다.
